# SAFE_WORDING.md

## 目的

このファイルは、アプリ内の文言が法律判断に見えないようにするための表現ルールです。

このアプリは、部屋の状態を記録・整理・共有しやすくするためのアプリです。

法律判断、責任判断、費用負担判断は行いません。

## 使ってよい表現

- 状態を記録する
- 写真を整理する
- 場所ごとに写真を残す
- 間取り図と写真を紐づける
- 入居時の状態を記録する
- 退去時の状態を記録する
- 確認漏れを減らす
- 共有しやすい形式にまとめる
- 管理会社や仲介会社に共有しやすくする
- PDF台帳を作成する
- 記録日時を残す
- コメントを残す
- 写真台帳を保存する

## 使ってはいけない表現

- 証拠になります
- 法的に有効です
- 退去費用を減らせます
- 請求を拒否できます
- 貸主負担です
- 借主負担です
- 原状回復費用を回避できます
- 裁判で使えます
- 必ず認められます
- トラブルに勝てます
- 請求されません
- 管理会社に対抗できます

## 注意書きの基本文

アプリ内の分かりやすい場所に、以下の注意書きを表示する。

```text
このアプリは法律判断を行いません。部屋の状態を記録・整理することを目的としています。
```

PDF台帳には以下の注意書きを入れる。

```text
この台帳は、部屋の状態を記録・整理することを目的としたものです。法律判断や費用負担の判断を行うものではありません。
```

## メール共有文面の例

件名:

```text
入居時の室内状態記録について
```

本文:

```text
お世話になっております。

入居時の室内状態について、確認した内容を写真台帳として整理しました。
念のため、記録として共有いたします。

添付のPDFをご確認ください。

なお、この台帳は室内状態を記録・整理する目的で作成したものです。

よろしくお願いいたします。
```

## ボタン文言の例

よい例:

```text
写真を追加する
場所を記録する
PDF台帳を作成する
共有用の文面をコピーする
入居時チェックリストを開く
状態を記録する
```

避ける例:

```text
証拠を作る
請求対策をする
貸主負担にする
退去費用を減らす
法的証拠を保存する
```
