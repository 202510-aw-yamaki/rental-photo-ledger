# 賃貸写真台帳メーカー

## これは何のアプリ？

**賃貸写真台帳メーカー**は、賃貸物件への入居時や退去時に、部屋の傷・汚れ・設備の状態を写真で記録し、あとから見返しやすい台帳としてまとめるためのアプリです。

引っ越しのときに、

- どこを撮ればいいのか分からない
- 撮った写真がバラバラで整理できない
- あとから「これはどこの写真だったか」が分からなくなる
- 管理会社や仲介会社に状態を共有しにくい
- 入居時の確認漏れが心配

といった困りごとを減らすことを目的としています。

## 主な使い道

このアプリは、次のような場面で使うことを想定しています。

- 入居前または入居直後の部屋の状態を記録する
- 退去前に部屋の状態を整理する
- 間取り図と写真を紐づけて、場所ごとに管理する
- 壁、床、水回り、エアコンなどのカテゴリーごとに写真を整理する
- コメント付きの写真台帳をPDFで出力する
- 管理会社や仲介会社へ、入居時の状態を共有しやすくする

## このアプリでできること

- 間取りPDFを読み込む
- 間取り図の上に番号や記号を配置する
- 番号ごとに写真を登録する
- 写真にカテゴリーを付ける
- 写真ごとに簡単なコメントを残す
- 登録日や作成日を含めたPDF台帳を作成する
- 入居時チェックリストで撮影漏れを減らす
- 共有用の文章を作成し、管理会社や仲介会社へ連絡しやすくする

## 想定している利用者

- 賃貸物件に入居する人
- 引っ越し前後に部屋の状態を記録したい人
- 写真を整理して残したい人
- 管理会社や仲介会社に状態を分かりやすく共有したい人
- 原状回復トラブルへの備えとして、記録を残しておきたい人

## 大切な注意

このアプリは、法律判断を行うものではありません。
このアプリで作成する台帳は、あくまでも部屋の状態を整理して記録するためのものです。

以下のような判断は行いません。

- 退去費用が妥当かどうか
- 貸主負担か借主負担か
- 請求を拒否できるか
- 法的に証拠として認められるか

目的は、部屋の状態を写真・場所・日付・コメントと一緒に分かりやすく残すことです。
個人を特定できる情報（物件名や住所など）は載せないでください。

## アプリの方向性

最初のバージョンでは、スマートフォンでも使いやすいWebアプリとして開発します。
インストール不要で使えること、写真を登録しやすいこと、PDFとして保存しやすいことを重視します。
